- <a href="https://docs.docker.com/engine/install/">Docker</a>
- 추가 예정

### Installation

아래의 순서대로 프로젝트를 구성해주세요

1. Clone the repo
   ```sh
   git clone https://github.com/yodafriend/webide.git
   ```
2. module 설치 및 husky 실행
   ```sh
   npm install
   ```
3. docker-compose 명령어 실행
   ```sh
   # 프로젝트 root 위치
   docker-compose up --build
   ```
4. 추가 예정
